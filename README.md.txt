### 🎫 Event Management

### 📢 Announcements

- View announcements
- Edit announcements
- Delete announcements

### 📚 Resources
- Add academic resources
- View available resources
- Edit resource details
- Delete resources

## 🗄️ Database Design

The system uses a **MySQL relational database** with the following tables:

### Tables
- `events`
- `announcements`
- `resources`

Each table includes:
- Primary key (`id`)
- Relevant fields
- Timestamp fields for record tracking
